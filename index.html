Monica

MONICA: MObile Neural voIce Command Assistant for mobile gaming

Abstract

Speech based user interface for mobile gaming is a growing need in the industry. While deep learning based automatic speech recognition (ASR) methods running inside mobile devices are making multiple breakthroughs, such as the ____, there is no concrete work in integrating such ASR inside a mobile gaming system. The biggest difficulty of ASR in mobile gaming is that users need an immediate response. This requires a real-time response speed, on the limits of the computational power of a mobile phone. In this demonstration, we present MONICA - a transformer based model that maintains near-SOTA performance, yet only uses a sixth of the memory and 3 times faster than the traditional model on a mobile device.

Size and Performance

The Monica model show similar recognition accuracy compared to the state of the art ASR methods, but with only ⅙ memory usage and 3 times faster on a mobile CPU.
